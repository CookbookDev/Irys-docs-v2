## Connecting to Irys

The Irys SDK reduces dependency bloat by providing dedicated packages for each token. Your import statements and connection code will differ depending on the token used for payment.

The following code is for using ethereum only, we [also have examples covering all of the tokens we support for payment](/build/d/sdk/import-configure). 

```ts
import Ethereum from "@irys-network/bundler-ethereum-node";
import NodeIrys from "@irys-network/bundler-client-node";

const getIrys = async () => {
  // RPC URLs change often, use a recent one from https://chainlist.org/
  const providerUrl = "";

  const irys = await new NodeIrys(Ethereum)
              .withWallet(process.env.PRIVATE_KEY)
              .testnet()
              .withRpc(providerUrl)
              .build();

  console.log(`Connected to Irys from ${irys.address}`);
  return irys;
};
```
