# `fund`

Funds your Irys account with the specified amount of tokens.

<Callout type="info" emoji="ℹ️">
Not all calls to `fund` will post immediately to your account, some blockchains are faster than others. 
</Callout>

## Parameters

- `-n`: The network to check. Use `testnet`.
- `-t`: The [token](/overview/supported-tokens) to use when funding.
- `-w`: Your [private key](/cli#using-private-keys).
- `--provider-url`: RPC URL to use. Required when using testnet.

## Example

``console
irys fund 1000000000000000 \
  -n testnet \
  -t ethereum \
  -w bf20......c9885307 \
  --provider-url https://rpc.sepolia.dev
```

<Callout type="info" emoji="ℹ️">
	RPC URLs change often, use a recent one from https://chainlist.org/
</Callout>
