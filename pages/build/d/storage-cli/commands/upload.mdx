# `upload`

Uploads a single file.

## Parameters

<ApiTable columns={[
	{ header: "Option", key: "type", span: 3 },
	{ header: "Description", key: "description", span: 9 },
]} data={[
	  { type: "-n", description: "The network to check, `testnet` or `devnet`, defaults to `testnet`." },
    { type: "-t", description: "The [token](/build/d/features/supported-tokens) to use when funding." },
    { type: "-w", description: "Your private key." },
    { type: "--tags", description: "[Tags](/build/d/features/tags) to include, format `<name>` `<value>`." },
    { type: "--provider-url", description: "RPC URL to use." },
]}/>


## Examples

### Testnet 

```console
irys upload myImage.png \
  -t ethereum \
  -w bf20......c9885307 \
  --tags tagName1 tagValue1 tagName2 tagValue2 \
```

### Devnet

```console
irys upload myImage.png \
  -n devnet \
  -t ethereum \
  -w bf20......c9885307 \
  --tags tagName1 tagValue1 tagName2 tagValue2 \
  --provider-url https://rpc.sepolia.dev
```

<Callout type="info" emoji="ℹ️">
	RPC URLs change often, use a recent one from https://chainlist.org/
</Callout>

## Downloading

Files uploaded via `irys upload` can be downloaded using the transaction ID provided after a successful upload.

```console
irys upload myImage.png \
  -n testnet \
  -t aptos \
  -w bf20......c9885307 \
  --tags tagName1 tagValue1 tagName2 tagValue2

Loaded address: 0xac568a981b1370b2e1baa8ce30bd5ac9e28c572d
Uploaded to https://gateway.irys.xyz/yPX6_rwxfJ_2gTvXSHGoumIvlFR-EwuRlUMmHpeLHUM
```
