# Downloading Data

Once data is uploaded, it becomes instantly accessible via our gateway. To download data, make a GET request from any public gateway.


**URL format**

`https://gateway.irys.xyz/[transaction-id]`

## Example

`https://gateway.irys.xyz/CO9EpX0lekJEfXUOeXncUmMuG8eEp5WJHXl9U9yZUYA`


<Callout type="info" emoji="ℹ️">
Irys is currently in private testnet. At mainnet launch, all uploaded data will be migrated from testnet to mainnet. Transactions IDs will not change. 
</Callout>
