# Importing & Configuring

The Irys SDK reduces dependency bloat by providing dedicated packages for each token. Your install statements, import and connection code will differ depending on the token used for payment.

Select the appropriate code from the options below when starting your project.

<div className="flex justify-center text-center mt-5">
[APTOS](#aptos) |  [Arbitrum](#arbitrum) | [Avalanche C-Chain](#avalanche-c-chain) | [Berachain](#berachain) | [BNB](#bnb) | [Chainlink](#chainlink) | [Eclipse](#eclipse) | [Ethereum](#ethereum) | [Base Ethereum](#base-ethereum) | [Linea Ethereum](#linea-ethereum) | [Scroll Ethereum](#scroll-ethereum) | [IoTeX](#iotex) | [Polygon](#polygon) | [Solana](#solana) | [USDC (on Ethereum)](#usdc-on-ethereum) | [USDC (on Polygon)](#usdc-on-polygon)

## Aptos

Installing:

```console
npm install @irys-network/bundler-aptos
```

Importing & Configuring:

```ts
import AptosBundlerIrys from "@irys-network/bundler-aptos";

const getIrysBundler = async () => {
  const irysBundler  = await AptosBundlerIrys().withWallet(process.env.PRIVATE_KEY);

  console.log(`Connected to Irys from ${irysBundler.address}`);
  return irysBundler;
};
```
