# `minHeight(height)`

Queries Arweave based on block height, returning all blocks at the specified height and after. Commonly used with [`maxHeight()`](./maxHeight).

## Parameters

-   `height`: Block height

```js
const results = await myQuery
	.search("arweave:blocks")
	.minHeight(1188272)
	.maxHeight(1188279);
```
