# `token(tokenName)`

Queries by token used for payment. Accepts [any token supported by Irys](/overview/supported-tokens).


## Parameters


- `tokenName`: The name of the [token](/overview/supported-tokens)

```js
const results = await myQuery
	.search("irys:transactions")
	.token("solana");
```
