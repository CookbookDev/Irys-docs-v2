# `price`

Checks the price to upload the specified number of bytes.

## Parameters

- `-t`: The [token](/build/data/features/supported-tokens) used for payment.
- `-n`: The network to check, use `testnet`.
- `--provider-url`: RPC URL to use. 

## Example

```console
irys price 1000000 \
  -t ethereum \
  -n testnet \
  --provider-url https://rpc.sepolia.dev
```

<Callout type="info" emoji="ℹ️">
	RPC URLs change often, use a recent one from https://chainlist.org/
</Callout>
