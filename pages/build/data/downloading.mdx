# Downloading Data

Once data is uploaded, it becomes instantly accessible via our gateway. To download data, make a GET request from any public gateway.


**URL format**

`https://testnet-gateway.irys.xyz/[transaction-id]`

## Example

`https://testnet-gateway.irys.xyz/CO9EpX0lekJEfXUOeXncUmMuG8eEp5WJHXl9U9yZUYA`
